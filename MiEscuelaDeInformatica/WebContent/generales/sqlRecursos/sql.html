SQL

El lenguaje SQL es el lenguaje estándar para la utilización de bases de datos relacionales. Es a la vez un potente lenguaje y fácil de aprender ya que es muy parecido al lenguaje natural. SQL se utiliza para controlar todas las funciones que un SGBD (Sistema Gestor de Bases de Datos) proporciona a sus usuarios:

  • Definición de datos: Permite definir la estructura y organización de los datos almacenados y de las relaciones entre ellos.
  • Recuperación de datos: Permite recuperar los datos almacenados en la base de datos.
  • Manipulación de datos: Permite añadir, modificar o borrar datos en una base de datos.
  • Control de acceso: Puede ser utilizado para restringir la capacidad de un usuario para recuperar, añadir, modificar o suprimir datos, protegiendo así los datos de accesos no autorizados.
  • Compartición de datos:SQL se utiliza para coordinar la compartición de datos por parte de usuarios concurrentes, asegurando que no interfieren unos con otros.
  • Integridad de datos:SQL define restricciones de integridad en la base de datos, protegiéndola de actualizaciones inconsistentes o fallos del sistema.

Lenguaje DDL. Creación y gestión detablas

Lenguaje DDL. Creación y gestión detablas

Permite definir la estructura y organización de los datos almacenados y de las relaciones entre ellos.

TABLA: unidad básica de almacenamiento compuesta de filas y columnas.

Sintaxis:
CREATE TABLE [NB_PROPIETARIO.] NB_TABLA
(NB_COLUMNA1 TIPO_DE_DATO [DEFAULT EXPRESIÓN] [NOT NULL]
[,NB_COLUMNA2 TIPO_DE_DATO [DEFAULT EXPRESIÓN] [NOT NULL]]
...);
tipo de dato: indica la naturaleza del dato. Ej. number (n,m), varchar2(n) ,date,
etc.
default indica el valor que tomará por defecto la columna si éste no es
especificado.
not null indica que la columna no puede tomar valores nulos.

Ejemplo

--Crea la tabla usuarios, (Oracle)
                    
CREATE TABLE USUARIOS
(ID_USUARIO NUMBER NOT NULL,
NOMBRE VARCHAR2(30),
FECHA DATE DEFAULT SYSDATE);

Consultas

Consultas

La sentencia SELECT permite recuperar la información almacenada en las tablas de la base de datos. Con esta sentencia podemos seleccionar (elegir registros de la tabla o tablas que deseemos), proyectar (seccionar las columnas que se deseen) y unir (extraer información de distintas tablas) .

Sintaxis:
SELECT [DISTINCT] { *|COLUMN [[AS] ALIAS][, COLUMN [[AS] ALIAS]]...} FROM NB_TABLA;
DISTINCT significa que las filas duplicadas solo se devolverán una vez.
Un asterisco (*) produce como resultado la visualización de todas las columnas de la tabla.
COLUMN: representa el nombre de una columna.
ALIAS sustituye en la cabecera de la sentencia SELECT el nombre de la columna.

Ejemplo

--Seleciona todas las columnas de la tabla empleados, (Oracle)
                    
SELECT * FROM EMPLEADOS;
SELECT ID_EMPLEADO, NOMBRE, FECHA FROM EMPLEADOS;
SELECT ID_EMPLEADO AS IDENTIFICADOR, NOMBRE, FECHA FROM EMPLEADOS;

Lenguaje de manipulación de datos. DML

Lenguaje de manipulación de datos. DML

Es una de las partes fundamentales del lenguaje SQL. El DML (Data Manipulation Language) lo forman las instrucciones capaces de modificar los datos de las tablas. Al conjunto de instrucciones DML que se ejecutan consecutivamente, se las llama transacciones y se pueden anular todas ellas o aceptar, ya que una instrucción DML no es realmente efectuada hasta que no se acepta (commit).

Inserción de registros en una tabla: INSERT.

Permite insertar una fila en una tabla, es decir añadir un registro de información en una tabla
Sintaxis:
INSERT INTO NB_TABLA [(COLUMNA[,COLUMNA] ... )]
VALUES (VALOR [,VALOR] ...);
Se puede omitir el nombre de las columnas siempre y cuando se respete el orden que tienen en la tabla:

Ejemplo

--Inserta una fila en la tabla departamentos, (Oracle)
INSERT INTO DEPARTAMENTOS VALUES (50,'DESARROLLO',2500);
