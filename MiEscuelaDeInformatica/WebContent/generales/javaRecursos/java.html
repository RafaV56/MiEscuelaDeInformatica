Java

Java es una tecnología que se usa para el desarrollo de aplicaciones que convierten a la Web en un elemento más interesante y útil. Java no es lo mismo que javascript, que se trata de una tecnología sencilla que se usa para crear páginas web y solamente se ejecuta en el explorador.

Identificadores

Identificadores

En Java, es necesario poner nombre a muchas cosas, como para las variables, los métodos y las clases. El nombre que utilizamos para estos elementos es lo que llamamos un identificador.

Ejemplo

public class Indentificadores {

	public static void main(String[] args) {
	//Ejemplo validos
	String x;
	String x1;
	String a0b1c3;
	
	//Invalidos
	
	//El indentificador no puede empezar con un número
	String 1x; 
	
	//No puede tener un espacio
	String mi edad; 
	
	//El asterisco no se puede usar en un identificador
	String a*3; 
	}

}

Variables

Variables

En Java, llamamos variable a un espacio para almacenar un valor que puede cambiar a lo largo del programa. Cada variable tiene asociado un nombre y un tipo. Cuando queremos utilizar el valor contenido en una variable, simplemente utilizamos su nombre.

Declaración e inicialización de variables

Una variable es un espacio de memoria al cual identificamos con un identificador.
En java, es necesario declarar las variables antes de utilizarlas.
Para declarar una variable, escribimos el tipo, seguido de un identificador.
int edad;
Se pueden declarar varias variables del mismo tipo, separándolas por comas.
double altura, peso;
La operación básica para dar valor a una variable es la asignación, denotada por el operador = altura=12.0;

Ejemplo

public class Variables {

	public static void main(String[] args) {
	
	
	//Cuando declaramos una variable debemos suponer que su valor inicial es indefinido,
	//con lo cual, no puede participar en ninguna otra operación.
								
	int x, y;

								
	y=x+1; //INCORRECTO:
	
	
	//Se puede dar valor inicial a una variable con una asignación. 
	//Y también se puede hacer en el momento de la declaración
								
	int x, y;
								
	x=3;
				
	y=x+1;

}

Tipos de datos

Tipos de datos

Java es un lenguaje estáticamente tipado. Eso significa se debe declarar explícitamente el nombre y el tipo de las variables antes de utilizarlas.

Primitivos, de valor

El lenguaje de programación Java da soporte a 8 tipos denominados primitivos, o de valor. Los tipos primitivos están predefinidos en el lenguaje y se nombran mediante una palabra reservada. Estos 8 tipos son:

byte
Entero con signo, entre -128 y 127. Almacenado en complemento a 2 con 1 byte (8 bits)
short
Entero con signo, entre -32,768 y 32,767 (inclusive). Almacenado en complemento a 2 con 2 bytes (16 bits)
int
Entero con signo, con valores entre -2,147,483,648 y 2,147,483,647 (inclusive). Para valores entero, éste es el tipo normalmente escogido, a menos que exista una buena razón para utilizar alguno de los otros. byte y short son más ahorrativos en la memoria o pueden ser utilizados por motivos de compatibilidad. long proporciona un rango más amplio.
long
Entero con signo, entre -9,223,372,036,854,775,808 y 9,223,372,036,854,775,807 (inclusive). En complemento a 2, utilizando 8 bytes (64 bits)
float
Éste tipo almacena un número con decimales en el formato de coma flotante, según el estándar IEEE754. Utiliza 32 bits para el almacenamiento. En general, no se utiliza a menos que necesitemos un ahorro de memoria o por motivos de compatibilidad.
double
Éste tipo almacena un número con decimales de doble precisión en el formato de coma flotante, según el estándar IEEE754. Utiliza 64 bits para el almacenamiento. En general, es el tipo utilizado cuando se necesitan valores que pueden tener decimales, pero ten en cuenta que siempre tiene pérdidas de precisión. Aunque nosotros lo utilicemos ampliamente mientras aprendemos a programar, en el mundo real nunca se utiliza para valores que deban ser absolutamente precisos, como valores monetarios, por ejemplo. El rango de números que puede almacenar va aproximadamente desde ± ~10-323.3 hasta ~10308.3. Tema 2
boolean
una variable booleana sólo puede contener dos valores: true y false. Es una variable de tipo lógico. Utilízala para representar condiciones que se puedan responder con verdadero y faso (o sí y no, si lo prefieres). Para representar un valor booleano basta con un bit, pero debido a la construcción de las memorias (en bytes), la cantidad exacta de memoria real utilizada no está definida. El nombre de éste tipo de datos está tomado del apellido de George Boole, que desarrolló la teoría de la lógica básica con dos valores (true y false).
char
El tipo de datos Char representa a un único carácter con la codificación Unicode. Ocupa 16 bits, y puede tomar valores entre '\u0000' (es decir, 0) y '\uffff' ( 65,535 inclusive).

Ejemplo

public class TiposDeDatos {

	public static void main(String[] args) {
	//Ejemplo validos
	
	byte unByte=127;
	byte otroByte=-128;
			
	short unShort=-32768;
	short otroShort=32767;
			
	int unInt=-2147483648;
	int otroInt=2147483647; 
			
	long unLong=-9223372036854775808L; //lleva la L al final
	long otroLong=9223372036854775807L; //lleva la L al final
			
	float unFloat=55.55F; //lleva la F al final
			
	double real=10.01234567890;
			
	boolean boleano=true;
	boolean boleano2=false;
			
	char unChar='\u0000'; //Representa al 0 y se encierra entre comillas simples
	}

}
