CSS

Hojas de Estilo en Cascada (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de documentos HTML o XML, esto incluye varios lenguajes basados en XML como son XHTML o SVG. CSS describe como debe ser renderizado el elemento estructurado en pantalla, en papel, hablado o en otros medios.

Sintaxis

Identificadores

El conjunto de Regalas CSS consta principalmente e un selector ejemplo el h2, luego entre llaves {} hacemos las declaraciones, que constan de 2 elementos, la propiedad y el valor, ejemplo color:red;

Ejemplo

/*
   Ejemplo de selector para una etiqueta h2 de HTML
*/
                    
h2{
   color:blue; /* La propiedad es color, y el valor es azul*/
   padding:0.5em;
}
